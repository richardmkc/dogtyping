# Audio Folder

Put your bark sound file here!

## Bark Sound
- Name it: `bark.mp3` or `bark.wav`
- Recommended format: MP3 or WAV

## Important
Once you add your audio file to this folder, make sure to:
1. Add it to git: `git add audio/`
2. Commit: `git commit -m "Add bark sound"`
3. Push: `git push`
