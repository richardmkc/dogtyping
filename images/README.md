# Images Folder

Put your dog and cow photos here!

## Dog Photos
- Name them: `dog1.jpg`, `dog2.jpg`, `dog3.jpg`, etc.
- Formats: `.jpg`, `.jpeg`, `.png`, `.gif` all work

## Cow Photos
- Name them: `cow1.jpg`, `cow2.jpg`, `cow3.jpg`, etc.
- Formats: `.jpg`, `.jpeg`, `.png`, `.gif` all work

## Important
Once you add your image files to this folder, make sure to:
1. Add them to git: `git add images/`
2. Commit: `git commit -m "Add dog and cow images"`
3. Push: `git push`
